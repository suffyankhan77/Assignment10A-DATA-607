#  Assignment10A-DATA-607

## Overview

This repository contains the complete work for **Assignment 10A – Sentiment Analysis with Text Mining in R** for **DATA 607**.

The objective of this assignment is to:

* Reproduce the sentiment analysis workflow from **Chapter 2 of *Text Mining with R***
* Extend the analysis using a different dataset (**IMDB Movie Reviews**)
* Incorporate an additional sentiment method (**syuzhet**)
* Compare results across multiple lexicons and datasets

##  Part 1 — Reproducing Chapter 2

The first part replicates the sentiment analysis workflow from:

> Silge, J., & Robinson, D. *Text Mining with R: A Tidy Approach*

Key steps:

* Load Jane Austen novels using `janeaustenr`
* Convert text into tidy format (one word per row)
* Apply sentiment lexicons:

  * **Bing** (positive/negative)
  * **AFINN** (numeric scores)
  * **NRC** (emotion categories)
* Analyze:

  * Joy words in *Emma*
  * Sentiment trends across novels
  * Lexicon comparison on *Pride & Prejudice*

---

##  Part 2 — Extension with IMDB Reviews

A new dataset was introduced:

* **IMDB Movie Reviews Dataset (sampled for GitHub size limits)**

Key steps:

* Load dataset via **raw GitHub link**
* Remove missing sentiment labels
* Tokenize reviews into tidy format
* Apply:

  * **Bing** → word-level sentiment
  * **AFINN** → document-level sentiment scores
  * **NRC** → emotion analysis

---

##  Part 3 — Additional Lexicon (syuzhet)

To extend beyond the original example:

* Used **syuzhet** for full-text sentiment scoring
* Compared with AFINN at review level

---

##  Part 4 — Comparison & Evaluation

* Combined **AFINN** and **syuzhet** scores
* Evaluated agreement with actual review labels

### Results:

* **AFINN agreement:** ~0.681
* **syuzhet agreement:** ~0.687

👉 Both methods effectively captured sentiment polarity.

---

##  Key Insights

* Sentiment analysis results depend on:

  * **Type of text** (narrative vs opinion)
  * **Lexicon used**
* Jane Austen novels:

  * Show gradual sentiment shifts across narrative
* IMDB reviews:

  * Show strong and direct sentiment patterns
* Different lexicons produce different interpretations
